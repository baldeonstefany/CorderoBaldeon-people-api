# CorderoBaldeon People API

API REST desarrollada con Java y Spring Boot para generar una lista de 10 personas consumiendo datos desde la API externa RandomUser.

## Descripción

Este proyecto consume la API pública RandomUser y expone un endpoint REST en formato JSON con los siguientes datos:

- Nombre completo
- Género
- Ubicación
- Correo electrónico
- Fecha de nacimiento
- Fotografía

El enfoque elegido fue **Back-end**, priorizando una API funcional, documentada y probada con Postman.

## Tecnologías utilizadas

- Java 21
- Spring Boot
- Maven
- Lombok
- Spring Web
- Postman
- Git / GitHub
